# What's New in v1.15.0

Date: 2026-01-29

## Highlights

- Add methods to retrieve subset columns and check for duplicate keys in ShapeShifter.
- Add performance profiling commands and documentation.
- Clean up whitespace and improve column name validation in UCanAccessSqlLoader.
- Enhance dependency graph structure and edge handling for 'provides' type.
- Add 'socken' column to site entity and define location entity with unnesting options.


## New Features

- Add methods to retrieve subset columns and check for duplicate keys in ShapeShifter.
- Add performance profiling commands and documentation.
- Add strict mode to Config for unresolved directive validation.
- Add support for uploading Excel files to data source directory.
- Add validation for duplicate keys and foreign key integrity in data validators.
- Enhance drop_duplicate_rows function to raise error for missing columns.
- Implement deferred linking optimization and tracking for foreign keys.
- Move filter registry and and exists_in filter implementation to transforms.
- Optimize data validation by implementing cache warmup for validators.


## Improvements

- Clean up whitespace and improve column name validation in UCanAccessSqlLoader.
- Enhance dependency graph structure and edge handling for 'provides' type.
- Enhance UnnestEditor with available columns for improved functionality.
- Improve expected column increase calculation in ForeignKeyConstraintValidator.
- Set ExcelWriter to manual calculation mode for performance improvements.
- Update entity type references from 'data' to 'entity' and enhance column hydration logic.


## Fixes

- Add 'socken' column to site entity and define location entity with unnesting options.
- Clean up whitespace and formatting across multiple files.
- Clear preview data and reset view mode when entity changes in EntityFormDialog.
- No grid-editor for new fixed value entities ([#182](https://github.com/humlab-sead/sead_shape_shifter/issues/182)).
- Remove outdated FIXME comment regarding config validation in ShapeShifter.
- Replace v-combobox with v-select for ID and Value Variables in UnnestEditor.
- Reset to form-only view when creating new entity.
- Update column selection in ShapeShifter to use get_subset_columns method.
- Update drop_duplicate_rows to return data instead of raising an error for missing columns.
- Update GraphNode and DependencyNode types to include additional entity types.
- Use router name-based navigation in HomeView.
